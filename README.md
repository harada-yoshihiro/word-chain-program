# 最長しりとり探索プログラム
本プログラムは、日本語文章を入力として受け取り、文章中に出現する名詞のみを抽出した上で、それらを用いた「最長しりとり列」を探索するプログラムです。
形態素解析によって名詞を正規化し、深さ優先探索を用いて、しりとりが成立する最大長の単語列を計算します。

## 目的
- 日本語文章を対象とした形態素解析の実践
- 抽出した語彙を用いたアルゴリズム設計・探索問題への応用

## 処理フロー
1. Janomeを用いて文章を形態素解析
2. 名詞のみを抽出し、読み情報を用いて正規化
3. 単語の形式を変換（読み, 表記, 先頭文字, 末尾文字）
4. 各単語間のしりとり接続関係を判定
5. 深さ優先探索（DFS）により、各単語を起点とした最長しりとり列を探索
6. 全体の中で最長となるしりとり結果を出力

## 工夫点
- 日本語特有の読み仮名処理に対応(読みが存在しない語は表記をカタカナ変換して補完)
- 名詞の重複を除去し、探索空間を削減
- 単語をノード、しりとり可能な関係をエッジとしたグラフとして扱う設計
- 再帰を用いた全探索アルゴリズムを採用

## アルゴリズム
- 探索手法：深さ優先探索（DFS）
- 各単語を一度しか使えない制約付き経路探索
- 最悪計算量：O(n!)（nは名詞数）

## 学んだこと
- 自然言語処理とアルゴリズム問題を組み合わせた設計
- 文学作品やニュース記事など異なる文章ジャンルでの挙動を確認できた
- データ前処理の重要性（正規化・重複除去）を実感した

## 今後の改善案
